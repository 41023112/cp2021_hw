Git 指令

1.git init = 建立新的本地端 Repository

2.git clone [Repository URL] = 複製遠端的 Repository 檔案到本地端

3.git status = 檢查本地端檔案異動狀態

4.git add [檔案或資料夾] = 將指定的檔案（或資料夾）加入版本控制。用 git add . 可加入全部

5.git commit = 提交（commit）目前的異動

6.git commit -m "提交說明內容" = 提交（commit）目前的異動並透過 -m 參數設定摘要說明文字

7.git stash = 獲取目前工作目錄的 dirty state，並保存到一個未完成變更的 stack，以方便隨時回復至當初的 state

8.git log = 查看先前的 commit 記錄

9.git push = 將本地端 Repository 的 commit 發佈到遠端

10.git push origin [BRANCH_NAME] = 發佈至遠端指定的分支（Branch）

11.git branch = 查看分支

12.git branch [BRANCH_NAME] = 建立分支

13.git checkout [BRANCH_NAME] = 取出指定的分支

14.git checkout -b [BRANCH_NAME] = 建立並跳到該分支

15.git branch -D [BRANCH_NAME] = 強制刪除指定分支（須先切換至其他分支再做刪除）

16.git reset --hard [HASH] = 強制恢復到指定的 commit（透過 Hash 值）

17.git checkout [HASH] = 切換到指定的 commit（與 git checkout [BRANCH_NAME] 相同)

18.git branch -m <OLD_BRANCH_NAME> <NEW_BRANCH_NAME> = 修改分支名稱
